Программа выводит данные о погоде в выбранных городах (или координатах) в выбранной дате
API ключ нужно регулярно менять в программе в файле main.

для запуска программы необходимо установить библиотеки requests и Flask и запустить файл app.py в ide
(желательно в pycharm) и перейти по адресу в строке вывода, затем следовать краткой инструкции на форме.
